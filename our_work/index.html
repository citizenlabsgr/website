Our Work

We use modern tools and practices to improve how decisions are made and services delivered. We use the following guidelines when working on projects:


• Design with users, not for them

• Invite participation and work in the open

• Use data to drive decisions

• Start small and continuously improve


Check out GitHub to see which projects are being actively worked on. Some highlights include:


• Mittens is a chat bot to check your voter registration status

• Ballot Preview lets you view and fill out annotated sample ballots

• Ballot Buddies is a web app to track your friends’ absentee ballots

• Elections API provides registration and ballot data to the above projects
